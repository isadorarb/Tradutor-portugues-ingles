# Tradutor de áudio Português-Inglês

Esse código implementa um tradutor que recebe um áudio de até 10 segundos em português gravado pelo usuário e devolve um áudio com a frase original traduzida para o inglês.

O código foi implementado em 4 etapas:
1. Gravação do áudio
2. Transcrição do áudio
3. Tradução da transcrição
4. Sintetização do texto traduzido em voz
